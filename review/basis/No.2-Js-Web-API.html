ECMA-262是用js语法标准 规定js的一些语法 w3c标准 是关于规定js怎么去用 主要包括 dom操作 bom操作 事件绑定 ajax请求 存储 W3C和ECMA-262没有关系，没有规定js基础相关的东西 不管什么变量类型、原型作用域和异步 只管定义浏览器js操作页面的api和全局变量 常说的js包含两部分 js基础知识(ECMA262)和js-web-api(W3C标准)
